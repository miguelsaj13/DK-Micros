# Donkey Kong ASCII

Proyecto 2 - Programación de Microprocesadores

## Compilar

g++ main.cpp menu.cpp game.cpp render.cpp input.cpp -o donkeykong -lncurses -lpthread


## Ejecutar

./donkeykong

## Librerías utilizadas

- pthread
- ncurses

## Controles

A -> izquierda
D -> derecha
W -> subir
S -> bajar
SPACE -> saltar
Q -> salir
